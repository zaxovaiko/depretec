depdetHunt down deprecated API usages

Scans your JS/TS project for @deprecated JSDoc symbols and maps each one to its replacement. Zero config, CI-ready.

View on GitHub

Zero config

Drop it in any project. Auto-respects .gitignore, detects tsconfig.json, skips @generated files. No setup required.

Source + deps

Scans both your source files and node_modules type definitions — finds deprecated usage no matter where the symbol comes from.

Replacement hints

Extracts replacement info from {@link X}, "Use X instead", "Replaced by X", "Prefer X" — human and machine readable.

Three output formats

Pretty table for humans, JSON for LLMs, Markdown for PR descriptions. Pipe anywhere.

CI-ready

Use --fail-on-found to block deploys when deprecated APIs are detected.

Programmatic API

Import scan() directly for custom tooling, scripts, or editor integrations.
